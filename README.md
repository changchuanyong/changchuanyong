## About Me

我主要关注机器人系统从底层控制到上层部署的完整实现，包括嵌入式控制、ROS2 通信与仿真、视觉定位、运动控制和系统联调。

当前研究方向是面向机器人插装与装配任务的 **力觉-视觉协同控制**，重点关注目标检测、位姿估计、接触状态感知与执行端修正。

## Featured Projects

### [ROS2 Mobile Robot](https://github.com/changchuanyong/ros2-mobile-robot)

基于 ROS2 的移动机器人仿真与可视化项目，包含 URDF / Xacro 机器人模型、Gazebo 世界、RViz 配置、激光雷达、相机与底盘相关配置。

### 仓储物流 AMR 小车系统

基于树莓派 4B + STM32F407 构建上下位机架构，完成 ROS2 功能部署、CAN 通信、自定义协议设计、状态监测、底盘联调与导航测试。

### 力觉-视觉协同引导的机器人充电接口对准

面向新能源汽车充电接口自动对接场景，完成目标检测、ROI 提取、孔位特征定位与位姿求解，支撑后续插装控制与协同修正。

### ROBOCON 机器人底盘电控系统

基于 STM32F4 + FreeRTOS 搭建实时控制架构，实现多电机协同控制、PID 闭环、传感器融合与复杂动作执行控制。
